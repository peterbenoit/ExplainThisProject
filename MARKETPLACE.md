Ever feel lost when you open a new codebase? "Explain This Project" is your personal guide. It scans your project, figures out what's inside, and writes a clear, simple `PROJECT_OVERVIEW.md` file for you.

It's like having a senior developer give you a tour of the project, but it only takes a few seconds.

## How It Works

It's as simple as 1-2-3:

1.  **Open any project** in VS Code.
2.  Run the **"Explain This Project"** command from the Command Palette (`Cmd+Shift+P`).
3.  **Done!** A new `PROJECT_OVERVIEW.md` file appears in your project root, ready to read.

The extension automatically detects:
*   Programming languages (JavaScript, Python, Rust, Go, and more)
*   Frameworks (React, Vue, Express, Django, Flask, etc.)
*   Dependencies and development tools
*   Key project files and directory structure

## Get Started

Install the extension, open a project, and give it a try!
